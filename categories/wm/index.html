tiling window manager 使用感想

tiling window manager 使用感想 前言 我之所以想使用 tiling window manager的原因: 一般的DE實在很肥。 想看看許多人所說，tiling window manager到底是如何的增加使用效率。 帥 另外這是我的使用規格: OS: Arch Linux x86_64 Host: B85M-DS3H Kernel: 5.4.93-2-lts Packages: 832 (pacman) Shell: zsh 5.8 WM: i3 CPU: Intel i5-4460 (4) @ 3.400GHz GPU: NVIDIA GeForce GTX 1650 SUPER Themes: Darcula 另外，所用設定全都放在 bloodnighttw/dotfile 心得文 使用兩週(大概啦)後，我覺得有幾點是我的想法： 其實讓操作變得有效率的不是Tiling window Manager 本身，而是快捷鍵本身。 手不用離開鍵盤其實是假的，但使用鍵盤的比率的確會使操作變快 調整視窗大小很方便，但彈性並沒有那麼高（但屬於夠用了）。 另外，其實i3wm並沒有想像中的難學，但對不願意花時間動手做的當然很難學，而他的設定檔也是算好懂的。 我的學習影片 https://www.youtube.com/watch?v=j1I63wGcvU4&list=PL5ze0DjYv5DbCv9vNEzFmP6sU7ZmkGzcf i3wm用了是真的會很習慣性的讓手放在鍵盤上，但其實只用鍵盤的機會很少，畢竟需要用到滑鼠操作的程式還是比較多，而且，有些程式用起來會很痛苦，像是 jetbrains toolbox，但那是少數，整體來講，用鍵盤操作視窗是相當的舒服的。 至於一些DE內會有的東西，單純的WM完全沒有，像是截圖軟體，這邊我使用flameshotk；處理通知的程式，這邊我使用dunst；Desktop Manager 則使用sddm；另外音訊的管理，我使用 Pavucontrol....
